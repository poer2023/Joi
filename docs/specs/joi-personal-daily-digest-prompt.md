# Joi 个人关注日报提示词

这是用于本机 Joi 定时自动化的生产提示词。它只处理公开网页，不读取或转发 Folo 中标记为私密的标题、URL、令牌或 Cookie。

```text
你正在执行“个人关注日报”自动任务。当前兴趣轮廓来自本机 Folo 的只读、脱敏快照，共 88 个订阅（50 个公开、38 个私密）。请只检索公开来源，并优先覆盖过去 24 小时真正有新增的内容。

覆盖面按下面八类轮换，不能长期只报 AI：
1. X / Twitter 与社交动态；
2. AI、模型与智能体；
3. 开发工具与软件；
4. 图片、摄影与视觉内容；
5. 视频、数码与硬件；
6. 资讯、博客与阅读；
7. 游戏与免费游戏/版本更新提醒；
8. 重要发布、插件与产品更新。

优先参考公开订阅家族：X/Twitter、Bilibili、YouTube、Telegram 公开频道、RSSHub 公共路由，以及公开 RSS/站点。具体订阅标题和 URL 只保存在本机生产任务中，不进入源码文档或可分享证据；不得猜测未核验内容。

必须调用 Joi 提供的只读网页工具。如果 MCP 工具被延迟暴露，先用 tool_search 搜索完整工具名 mcp__joi_web__web_search 和 mcp__joi_web__web_extract；不得在没有搜索工具目录前声称工具不存在。先调用 mcp__joi_web__web_search；对准备推送的至少一条结果再调用 mcp__joi_web__web_extract 核验正文或官方页面。只保留发布时间可核验、链接可访问、对我有实际价值的条目。若某类过去 24 小时没有可靠更新，写“本期无可靠新增”，不要用旧闻凑数。

Telegram 输出要求：简体中文；先给“今天最值得看”的 1-2 条，再按类别给最多 8 条；每条包含标题、1 句价值判断和原始 URL；总长控制在 3000 字符以内；不输出工具过程、隐私数据、追踪参数或重复链接。每周轮换弱覆盖来源，尽量在七天内覆盖全部八类兴趣。
```
