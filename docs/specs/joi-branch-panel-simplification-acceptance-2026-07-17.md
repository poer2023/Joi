# Joi 分支面板精简验收合同

## Scope

- Target: 安装版 `/Applications/Joi.app` 的聊天顶栏与右侧「分支」Tab
- Keep: 当前会话的分支关系、新建分支、版本信息编辑、上下文压缩、导入导出
- Remove: 顶栏独立分支按钮、Tab 内重复页头、手动刷新、重复空状态与常驻维护表单
- Style boundary: 只复用现有间距、颜色、圆角、字号和按钮样式，不改变 Joi 的 UI 视觉语言

## Information Structure

- 顶栏只保留统一的右侧检查器开关
- 分支只从右侧检查器的「分支」Tab 进入
- 首屏只显示当前会话范围、版本关系、分支状态与「从这里新开分支」
- 名称、说明、上下文压缩和迁移操作统一放进默认折叠的「高级」区域

## Interaction Rules

- 打开「分支」Tab 时自动读取当前会话的分支，不要求用户手动刷新
- 单节点只出现一次「尚无其他分支」状态，不再额外渲染空状态卡
- 新建分支无需先填写名称；默认名称由现有分支能力生成，原会话保持不变
- 版本命名与说明仍可在「高级」中编辑并保存
- hover、focus、active、disabled 与滚动行为继续沿用当前组件约定
