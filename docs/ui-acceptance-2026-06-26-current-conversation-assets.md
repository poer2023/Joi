# UI Acceptance Contract

## Scope

- Project: Joi Desktop browser preview
- Target URL or app screen: `http://127.0.0.1:5173/`, chat view right inspector `资产`
- Files likely to change: `apps/joi-desktop/frontend/src/App.tsx`, `apps/joi-desktop/frontend/src/styles.css`
- User job: assets means only files/images uploaded in the current conversation or files/images generated by the current conversation.

## Reference

- Primary reference: current right inspector Assets tab behavior.
- What to copy: compact list density and existing inspector visual style.
- What not to copy: global artifact inventory, other rooms, other threads, project-wide assets, memory/task/system records.

## Information Structure

- Must keep: one compact Assets tab.
- Must add: separate uploaded vs generated rows when present.
- Must remove: unrelated global artifacts from other conversations.
- Must not add: project asset manager, global library, storage/schema changes.

## Visual Rules

- Layout density: compact, scan-friendly rows.
- Spacing: use existing `asset-mini-list` and `asset-mini-row` scale.
- Typography: small metadata line, no large explanatory copy.
- Color: existing neutral palette.
- Borders/shadows: reuse existing row border/background.

## Interaction Rules

- Required interactions: switching to Assets shows only current conversation assets.
- Empty state: say the current conversation has no uploaded or generated assets.
- Mobile behavior: no new mobile-specific behavior.

## Verification

- Browser target: `http://127.0.0.1:5173/`
- Viewports:
  - Desktop: current in-app browser viewport
- Screenshot: `docs/ui-acceptance-2026-06-26-current-conversation-assets.png`
- DOM checks:
  - `#right-inspector-assets .asset-mini-row` count is `0` for the current `开发测试` conversation.
  - Empty state is `开发测试 暂无上传或生成资产。`
  - The tab is selected with `aria-selected="true"`.
- Console/network requirements: no browser console errors.
- Commands:

```bash
pnpm --dir apps/joi-desktop/frontend build
```

- [x] Assets tab is scoped to current conversation.
- [x] Uploaded message attachments appear as assets.
- [x] Generated artifacts appear only when their source conversation/message/run belongs to the current conversation.
- [x] Global/project/thread inventory is not shown as assets.
